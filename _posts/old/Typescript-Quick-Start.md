타입스크립트를 설치했다면 타입스크립트 컴파일러인 tsc를 이용해 컴파일을 할 수 있습니다.
hello.ts 파일을 명령 프롬프트에서 컴파일하고 실행하기
``` bash
    tsc hello.ts    <-컴파일
    node hello.js   <-실행
```
이것 말고도 다양한 명령어들이 존재합니다.
